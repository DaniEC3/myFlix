Get data about a single movie by title /movies/[Name] GET None
A JSON object holding data about a single movie, containing an ID, name,description,genre, director and year released. Example:
{
"_id": ObjectId,
"name":"The Godfather",
"description": "An organized crime dynasty's aging patriarch transfers control of his clandestine empire to his reluctant son.",
"genre": ObjectId,
"director": ObjectId,
year_released: ISODate('1972-03-24T00:00:00.000Z'),
"imagePath": "/images/default.jpg"
}
